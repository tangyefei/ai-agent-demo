## 大模型 vs AI SDK Runtime 职责分工

| 职责 | 执行者 |
|------|--------|
| 理解用户意图 | **大模型** |
| 生成工具调用指令 | **大模型** |
| 提取并输出参数（如 `location: "San Francisco"`） | **大模型** |
| 判断调用哪个工具 | **大模型** |
| 校验参数是否符合 Schema | **AI SDK Runtime** |
| 调用 execute 函数并返回结果 | **AI SDK Runtime** |
| 管理循环、控制迭代次数 | **AI SDK Runtime** |
| 决定何时停止 | **大模型** + SDK 配置 |

## 简化理解

```
大模型   →  "决策者"  (做什么? 参数是什么?)
SDK Runtime → "执行者"  (调用代码、管理循环、校验)
```

**一句话**：大模型负责"想清楚"，SDK Runtime 负责"做清楚"。
